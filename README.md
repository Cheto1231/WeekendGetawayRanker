# Weekend Getaway Ranker

A Python project that ranks weekend travel destinations in India based on distance, rating, and popularity.

## Technologies
Python, Pandas

##Datasets
https://www.kaggle.com/datasets/saketk511/travel-dataset-guide-to-indias-must-see-places

## Files
- weekend_getaway_ranker.py
- travel_dataset.csv
- requirements.txt
- sample_output.txt

## Run
pip install -r requirements.txt  
python weekend_getaway_ranker.py
